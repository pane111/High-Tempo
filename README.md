# High Tempo
 A game about going very fast
 

# CONTROLS:


W - Move forward

S - Move backwards

Space - Jump

Space in midair - Dash

Right click (hold) - Shoot (Release when drone is fully charged to perfrom laser beam)

Left click - Attack

Left Shift - Backstep

A/D - Sidestep
